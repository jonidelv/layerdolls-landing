# Layerdolls Landing Page

Static landing page for Layerdolls bookmark manager.

Hosted at [layerdolls.com](https://layerdolls.com)

## Tech Stack

- **HTML** - Static page
- **CSS** - Inline styles
- **No build step** - Just open in browser

## Development

Simply open `index.html` in a browser or serve with any static server:

```bash
# Python
python -m http.server 3000

# Node.js
npx serve .
```
